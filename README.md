 # trainer-api

RESTful web API for the Trainer App.

## Install
```
npm install
```

## Run
```
npm start
```

## Documentation
http://localhost:4000

### Predefined Users
There are 5 predefined users in the API:

| username | password | role |
| --- | --- | --- |
| admin | 1234 | admin |
| user1 | 1234 | default |
| user2 | 1234 | default |
| user3 | 1234 | default |
